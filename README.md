# CricManiac

## Overview
CricManiac is a WebApplication which provides users the result of ICC World Cup 2019 and future matches,
news related to the sports in a pleasant way.

## Built With
- Love <3
- Node.js
- Vanilla JS
- Express
- EJS
- Request
- CricApi
- NewsApi
